
### INSTALLATION

```
INSTALLATION
    ./sentinel-node.sh [options] [instruction]
    ./sentinel-node.sh [wireguard|v2ray|spawner] [install|remove]
Deploy Wireguard
    ./sentinel-node.sh wireguard install
Deploy V2Ray
    ./sentinel-node.sh v2ray install
Deploy spawner
    ./sentinel-node.sh spawner install
```
